.addClass(className)


Description

Adds the specified class(es) to each element in the set of matched elements.

This method does not replace a class. It appends it to the given classes of the node collection.


Uses

HTML

<div id="box" class="">
  Text will turn red
</div>

CSS

#box {
  width: 50px;
  height: 50px;
  border: 1px solid black;
}

.red {
  color: red;
}

jaysQuery

$j('#box').addClass('red');
Text will turn red

What is jaysQuery?

jaysQuery is a small yet versatile JavaScript library. It was created by Justin Cox in 2018 to assist with DOM traversal and manipulation, event-handling, animation, and making AJAX requests.
